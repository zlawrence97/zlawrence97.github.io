Project 3 - Stereo Hybrid

by: Zuri Lawrence

Implementation

Instead of having the input be an already rectified image, I created the rectified image. First I found the SURFFeatures of each image, then matched those two images with their 30 strongest points. The result of aligning those images was the rectified image. Using this image I was able to compute the depth map of the image.
This process was done for two additional sizes (which were difficult to select because I wanted to make sure the dimensions were whole numbers so I had to find factors of the original width and height.)
